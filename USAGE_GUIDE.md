## ğŸ”§ Troubleshooting

### Problem: "Password incorrect"
**Solution**:
- Check if CAPS LOCK is on
- Password is case-sensitive
- Contact IT admin for correct password

### Problem: "No student found"
**Solution**:
- Verify you're entering **Reg. No** (not St. ID)
- Check if student exists in the Excel file
- Make sure Excel file uploaded correctly

### Problem: "Error reading Excel file"
**Solution**:
- Make sure it's the correct Excel file
- File should have school name in first row
- Column headers in second row
- Try uploading again

### Problem: "Error generating certificate"
**Solution**:
- Make sure Word template uploaded correctly
- Template should have placeholders: {{NAME}}, {{REGNO}}, etc.
- Try uploading the template again

### Problem: Files uploading slowly
**Solution**:
- Check your internet connection
- Large files take longer
- Wait for upload to complete
- Don't refresh during upload

### Problem: App not loading
**Solution**:
- Check internet connection
- Try different browser
- Clear browser cache
- Contact IT admin
